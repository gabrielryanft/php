Math

Ordem de precedência

  • () Parenteses
  • ** Potência
  • * Multiplicação |-| / Divisão |-| % Resto de divisão(Modulus(en))

  • - Subtração |-| + Soma

Funcões:

  • abs() - Absolute(en): absoluto ("Versão" positive de um número.)
  • base_convert() - Convertor de Base: converte a base númerica.
  • funções de arredondamento:
    • ceil() - Arredonda para cima
    • floor() - Arredonda para baixo
    • round() - Arredondamento matemático
  • hypot() - Calcula a hipotenusa
  • intdiv() - Divisão inteira
  • min e max:
    • min() - Retorna o menor número de uma sequência
    • max() - Retorna o maior número de uma sequência
  • pi() - Retorna o valor de pi. ()Também da para usar a constante: M_PI (Já que ala é uma constante, é tudo em letras maiusculas.)
  • pow() - Power(en): Potência. (ou usa o **. (Esse pow existe porque o ** não existia antes, então usavam esse.)) quando você ultiliza a função pow() você perde a ordem de precedência.
  • Funcões trigonometricas:
    • sin() - calcula o seno
    • cos() - calcula o cosseno
    • tan() - calcula a tengente
  • Também existem funções para converter graus(degree(en)) para rediano.
  • sqrt() - calcula a raiz quadrada de um número. Ou você pode usaro o: "x ** (1/2)" (Obviamente, o "x" é o número que você quer a raiz quadrada.) (para raiz cubica: x ** (1/3) para raix quadrupla (1/4) e etc.)

Sinal "+" no php:

No php, o sinal + (mais) é somente usado no sentido matemático! Ele não concatena nada!
